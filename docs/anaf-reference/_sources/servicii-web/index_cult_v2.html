Folosirea serviciului este supusa urmatoarelor reguli:

ааааааааааа a) Un request poate contine maxim 500 de CUI-uri. Un client poate executa maxim 1 request pe secunda.

 

ааааааааааа b) Orice tentativa de suprasolicitare a serverului va fi pedepsita conform reglementarilor in vigoare.
